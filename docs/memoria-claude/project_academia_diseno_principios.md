## Los 3 pilares del diseño del sitio web

El ojo humano percibe en este orden de prioridad:
1. **Luz** — brillo, contraste, iluminación
2. **Sonido** — feedback auditivo, música, efectos
3. **Colores y movimiento** — animaciones, transiciones, paleta viva

**Why:** Jhony lo definió explícitamente como directriz de diseño para Academia Da Vinci IA. El sitio es para niños 9-13 y necesita estimulación constante.

**How to apply:** Toda decisión de diseño, nueva sección, componente o animación debe pasar por este filtro:
- ¿Tiene suficiente luz/brillo/glow? (fondos con gradientes luminosos, ojos de Robotsin brillando, efectos de glow en botones)
- ¿Hay feedback de sonido? (juice.js GCSound ya tiene wins/clicks — asegurarse de que esté activo en toda interacción clave)
- ¿Hay color vivo y movimiento? (animaciones CSS, partículas, scroll-reveal, tilt effects, el reel de videos)

**Nunca diseñar secciones estáticas, apagadas o silenciosas.** Cada elemento debe sentirse vivo.
